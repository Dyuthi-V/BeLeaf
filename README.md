# BeLeaf

## Transforming Everyday Actions into Meaningful Environmental Impact

BeLeaf is a gamified sustainability platform designed to encourage consistent environmental action through interactive challenges, artificial intelligence, and community-driven engagement.

The platform addresses a growing global issue: while environmental awareness is increasing, many individuals struggle to translate concern into meaningful daily action. BeLeaf simplifies sustainability by breaking environmental responsibility into small, achievable tasks that users can integrate into their everyday lives.

Through gamification, AI-assisted guidance, progress tracking, and collaborative challenges, BeLeaf creates an engaging ecosystem where users are rewarded for positive environmental behavior while contributing toward measurable collective impact.

# Overview

Environmental sustainability is often perceived as complex, inconvenient, or difficult to begin. Many people are willing to contribute positively toward climate action but lack awareness of simple and effective ways to participate.

BeLeaf bridges this gap by providing users with:

* Daily sustainability challenges
* Reward-based progression systems
* AI-generated eco guidance
* Community participation features
* Environmental impact tracking
* Interactive visual growth systems

The platform transforms environmental responsibility into an engaging and habit-forming experience.

---

# Problem Statement

Research consistently indicates a significant gap between environmental concern and environmental action.

Although a majority of individuals acknowledge the importance of climate responsibility, only a small percentage consistently engage in sustainable behaviors. This disconnect is primarily caused by:

* Lack of practical guidance
* Limited motivation
* Absence of measurable feedback
* Difficulty maintaining long-term habits
* Minimal community engagement

Environmental awareness alone is insufficient without systems that encourage sustained participation.

BeLeaf addresses these challenges by converting sustainability into manageable, rewarding, and socially engaging activities.

---

# Proposed Solution

BeLeaf is a gamified web platform centered around the concept of “everyday environmentalism.”

The platform encourages users to complete environmentally responsible tasks, referred to as “Side Quests,” which range from simple daily actions to collaborative community initiatives.

Users earn leaves, experience points (XP), streaks, and achievements as they complete tasks. These rewards contribute toward the visual growth of a digital tree and eventually an entire forest ecosystem, reinforcing progress and long-term engagement.

The platform also integrates an AI-powered assistant named “Leafy,” which provides sustainability recommendations, answers user questions, and helps users discover personalized environmental actions.

Additionally, BeLeaf incorporates environmental news updates and collaborative community challenges to connect individual actions with broader global impact.

## Installation and Setup

Follow these steps to clone the repository and run the application locally.

1. Clone the repository
git clone [https://github.com/Dyuthi-V/BeLeaf.git]

2. Navigate into the project folder
cd BeLeaf

3. Install dependencies using the legacy peer flag
npm install --legacy-peer-deps

4. Start the local development server
npm run dev

5. Open the application
Navigate to http://localhost:5173/ in your browser
